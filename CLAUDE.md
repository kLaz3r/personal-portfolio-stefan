## Project Overview

Modern personal portfolio website built with Next.js 16.2.1, TypeScript, and Tailwind CSS 4.2.2, featuring animated UI, project showcase, and automated photo gallery management.

## Development Commands

```bash
# Install dependencies
pnpm install

# Start dev server with Turbopack (uses port 3000)
pnpm run dev

# Build production bundle
pnpm run build

# Start production server
pnpm start

# Run linting
pnpm run lint

# Generate graphics gallery data
pnpm run generate-graphics

# Photo gallery automation (requires Python + Pillow)
pip install Pillow
python generate-image-array.py
```

**Note**: Dev server runs on `http://localhost:3000` by default. `predev` and `prebuild` hooks automatically run the graphics data generation script.

## Project Structure

- `src/app/` - Next.js App Router (pages, layouts, global CSS)
- `src/app/business/` - Quote wizard for business card QR code
- `src/components/` - React components (prefer relative imports `../data`, not `@/data`)
- `src/components/ui/` - shadcn/ui components
- `src/contexts/` - React contexts (ThemeContext, LanguageContext)
- `src/data/` - Static data files (auto-generated photos.ts, manual projects.ts)
- `public/images/` - Photo gallery source images
- `public/mockups/` - Project showcase images

## Architecture Details

### Next.js Configuration

- App Router with TypeScript strict mode
- Path alias: `@/*` maps to `./src/*`
- Image optimization disabled (see `next.config.ts`)
- Custom allowed dev origin: `192.168.3.9`
- Security headers configured for all routes

### Styling System

- Tailwind CSS 4.2.2 with `@theme inline` blocks
- CSS variables: `--brand-primary`, `--text-secondary`, `--background-secondary`, `--background-tertiary`
- Font: Montserrat (loaded via next/font in layout.tsx)
- shadcn/ui with `radix-maia` style and `hugeicons` icon library

### Animation Patterns

Uses `motion` library (NOT framer-motion):

- Spring transitions with `anticipate` or `easeInOut` easing
- `whileInView` for scroll-triggered animations
- `viewport={{ once: false }}` for re-triggering on scroll
- `active:scale-90` for button press feedback

### Component Conventions

- Arrow function components: `export const Component = (): JSX.Element =>`
- Props: `type Props = { ... }` pattern
- Image optimization: `quality={75}`, `loading="lazy"`, `placeholder="blur"` with base64

### Data Generation

**Photo Gallery** (`src/data/photos.ts`):

- Auto-generated by `generate-image-array.py`
- Scans `public/images/` → calculates aspect ratios
- Run after adding/removing images

**Graphics Gallery** (`src/data/graphics-projects.ts`):

- Auto-generated by `scripts/generate-graphics-data.ts`
- Scans `public/mockup_graphics/` directories
- Runs automatically before `dev` and `build`

## Business Page (/business)

Multi-step quote wizard for business card QR codes. Flow:

```
Landing → Service Select → Questions (6 per branch) → Summary → WhatsApp
```

- **Design branch**: project type, status, style, timeline, budget, notes
- **Web branch**: project type, design status, features, timeline, budget, notes
- Supports sequential "Both" flow with Part 1/2 indicators
- WhatsApp wa.me link with pre-formatted message
