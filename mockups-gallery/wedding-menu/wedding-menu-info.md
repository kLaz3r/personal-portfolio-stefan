# Wedding Menu - Elegant Menu Design

## Project Details

**Type:** Wedding Menu Design
**Dimensions:** A4 (210 x 297 mm) bi-fold to A5 (210 x 148 mm)
**Finish:** Premium wedding stationery

## Description

Elegant bi-fold wedding menu design. A4 format folds to A5, creating a sophisticated presentation for wedding receptions. The menu features a refined layout suitable for formal dining occasions, with elegant typography and design elements that complement wedding aesthetics. The bi-fold format provides ample space for menu courses while maintaining an elegant, compact presentation for each guest setting.

## Tools Used

- CorelDRAW (Design & Layout)
