# Jina Sibiului Shop - Header Design

## Project Details

**Client:** Jina Sibiului Shop
**Type:** Shop Header / Signage
**Dimensions:** 1000 x 2400 mm
**Finish:** Large format print for shop front display

## Description

Large format shop header design for Jina Sibiului retail store. The design serves as the main storefront signage, featuring bold typography and visual elements to attract customers and establish brand identity. The vertical format is optimized for building-mounted display, creating a prominent street-level presence for the shop.

## Tools Used

- CorelDRAW (Design & Layout)
